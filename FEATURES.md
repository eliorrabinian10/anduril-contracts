# 🎯 תכונות המערכת - Anduril Contracts Dashboard

## 🏠 דשבורד ראשי (Dashboard)

### כרטיסי סיכום
- **📊 סה"כ חוזים**: מספר החוזים הכולל במערכת
- **💰 סה"כ שווי חוזים**: שווי כל החוזים ביחד (בפורמט מקוצר: B/M/K)
- **📈 ממוצע לחוזה**: השווי הממוצע של חוזה בודד
- **🏛️ גורמים ממשלתיים**: מספר הגופים הממשלתיים השונים

### גרפים וויזואליזציות
1. **חוזים לפי שנים** (Line Chart)
   - מעקב אחר שווי החוזים לאורך זמן
   - זיהוי מגמות והתפתחות

2. **מספר חוזים לפי שנים** (Bar Chart)
   - כמות החוזים בכל שנה
   - זיהוי שנות שיא

3. **התפלגות לפי גורמים ממשלתיים** (Horizontal Bar Chart)
   - Top 7 גורמים ממשלתיים
   - השוואת שווי חוזים
   - צבעים שונים לכל גורם

4. **חלוקה יחסית** (Pie Chart)
   - אחוזי התפלגות בין גורמים
   - תצוגה גרפית של נתח שוק

5. **טבלת סיכום**
   - נתונים מפורטים לכל שנה
   - מספר חוזים, שווי כולל, ממוצע
   - מיון ואפשרויות סינון

### אינטראקטיביות
- ✅ רענון נתונים בלחיצת כפתור
- ✅ Tooltips מפורטים בכל גרף
- ✅ עיצוב responsive למכשירים שונים
- ✅ אנימציות חלקות

## 📋 רשימת חוזים (Contracts List)

### תצוגת כרטיסים
כל חוזה מוצג בכרטיס נפרד עם:
- **שם מקבל החוזה**: Anduril Industries או שם אחר
- **סכום החוזה**: בצבע בולט עם פורמט מקוצר
- **מזהה חוזה**: Award ID ייחודי
- **גורם ממשלתי**: הארגון המממן
- **יחידה**: Sub-agency (אם קיים)
- **סוג חוזה**: Award Type
- **תאריכים**: התחלה וסיום
- **תיאור**: פירוט החוזה (אם זמין)

### מיון מתקדם
- 💰 **לפי סכום**: מהגבוה לנמוך או להיפך
- 📅 **לפי תאריך**: מהחדש לישן או להיפך
- 🏛️ **לפי גורם ממשלתי**: מיון אלפביתי

### פיצ'רים נוספים
- ✅ Pagination - ניווט בין 50 חוזים בכל עמוד
- ✅ Hover effects - הדגשה בעת מעבר עכבר
- ✅ עיצוב כרטיסים מודרני
- ✅ תמיכה ב-RTL (עברית)

## 📈 ניתוח פיננסי (Analytics)

### כרטיסי תובנות (Insights)
1. **📈 שיעור צמיחה כולל**
   - חישוב אחוזי צמיחה מתחילת התקופה
   - אינדיקטור להתפתחות החברה

2. **🎯 טרנד נוכחי**
   - ניתוח 3 השנים האחרונות
   - סיווג: צמיחה חזקה / צמיחה / יציבות / ירידה
   - אייקונים ויזואליים

3. **🔮 תחזית לשנה הבאה**
   - חיזוי שווי חוזים בהתבסס על מגמה קיימת
   - אלגוריתם linear regression פשוט

4. **💼 חוזים פעילים**
   - מספר כולל של חוזים במערכת

### גרפים מתקדמים

1. **Year-over-Year Growth**
   - שיעור צמיחה שנתי באחוזים
   - זיהוי שנות צמיחה/ירידה
   - Bar chart צבעוני

2. **שווי מצטבר לאורך זמן**
   - Area chart עם gradient
   - הצגת מגמת צמיחה כוללת

3. **ממוצע שווי חוזה לפי שנה**
   - Line chart
   - זיהוי שינויים בגודל החוזים הממוצע

4. **מגמת מספר החוזים**
   - Area chart
   - מעקב אחר כמות החוזים

### טבלת סטטיסטיקות מפורטת
- רשימת Top 10 גורמים ממשלתיים
- **עמודות**:
  - גורם ממשלתי
  - מספר חוזים
  - שווי כולל
  - ממוצע לחוזה
  - אחוז מסה"כ

### חישובים אוטומטיים
- ✅ Growth Rate calculation
- ✅ Trend analysis
- ✅ Average calculations
- ✅ Predictive analytics

## 🔍 חיפוש מתקדם (Advanced Search)

### פילטרים זמינים

1. **מילות חיפוש**
   - חיפוש מרובה (מופרד בפסיקים)
   - ברירת מחדל: Anduril
   - דוגמאות: Defense, Technology, Surveillance

2. **טווח תאריכים**
   - תאריך התחלה
   - תאריך סיום
   - Date pickers נוחים

3. **סינון לפי סכום**
   - סכום מינימלי ($)
   - סכום מקסימלי ($)
   - לא חובה - השאר ריק לכל הסכומים

### תוצאות חיפוש

**סיכום תוצאות**:
- מספר החוזים שנמצאו
- שווי כולל של התוצאות
- כפתור ייצוא ל-CSV

**כרטיסי תוצאות**:
- מידע מלא על כל חוזה
- עיצוב דומה לרשימת החוזים
- תיאור מקוצר (150 תווים ראשונים)

### ייצוא נתונים
- 📥 **ייצוא ל-CSV**
  - כל תוצאות החיפוש
  - פורמט Excel-friendly
  - כולל כותרות
  - שם קובץ עם תאריך

### חוויית משתמש
- ✅ חיפוש real-time
- ✅ הודעות על חוזים שלא נמצאו
- ✅ איפוס מהיר של הפילטרים
- ✅ טעינה עם אינדיקטור

## 🔧 תכונות טכניות

### Backend API
- **Caching חכם**: 1 שעה cache למניעת עומס
- **Error handling**: טיפול בשגיאות ברמה גבוהה
- **CORS enabled**: תמיכה ב-cross-origin
- **RESTful design**: API סטנדרטי ונקי

### Frontend
- **React 18**: טכנולוגיה מודרנית
- **React Router**: ניווט client-side
- **Recharts**: ויזואליזציות איכותיות
- **Responsive design**: תמיכה בכל המכשירים
- **RTL support**: ממשק בעברית

### ביצועים
- ⚡ Fast loading times
- ⚡ Optimized re-renders
- ⚡ Lazy loading של נתונים
- ⚡ Client-side caching
